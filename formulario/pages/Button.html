Elementos de buttons

Inicio Inputs Contacto Buttons
  1. type : Define el tipo de botón (`submit`, `reset`, `button`).
  2. disabled : Desactiva el botón, impidiendo la interacción.
  3. name : Define un nombre para el botón, útil en formularios.
  4. value : Especifica un valor asociado al botón.
  5. autofocus : Hace que el botón reciba el foco automáticamente al cargar la página.
  6. form : Asocia el botón a un formulario específico mediante su `id`.
  7. formaction : Especifica la URL a la que se enviará el formulario cuando se haga clic en el botón.
  8. formenctype : Define cómo se codifican los datos del formulario (`application/x-www-form-urlencoded`, `multipart/form-data`, `text/plain`).
  9. formmethod : Especifica el método HTTP para enviar el formulario (`GET`, `POST`).
  10. formnovalidate : Indica que el formulario no debe validarse al enviarse.
  11. formtarget : Define dónde se abrirá el resultado del envío (`_self`, `_blank`, `_parent`, `_top`).
  12. class : Permite asignar clases CSS al botón.
  13. id : Define un identificador único para el botón.
  14. style : Permite aplicar estilos en línea.
  15. title : Muestra un texto emergente cuando el usuario pasa el cursor sobre el botón.
  16. data-* : Permite definir atributos personalizados para almacenar datos.
