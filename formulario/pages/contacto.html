propiedades action

  1. action : Define la URL donde se enviarán los datos del formulario.
  2. method="post" : Usa el método POST para enviar datos al servidor.
  3. enctype="multipart/form-data" : Permite enviar archivos en el formulario.
  4. input type="file" : Permite seleccionar archivos.
  5. accept="image/*,video/*,audio/*" : Filtra los tipos de archivos permitidos.
