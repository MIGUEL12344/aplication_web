los input son los elementos que se crean como contenido de un formulario, y nos permiten interaqctuar con el usuario, recolectar informacion a traves de nuestra pagina

TIPOS DE INPUT

  • text: Permite escribir texto normal en un campo.
  • password: Similar al tipo text, pero oculta los caracteres ingresados.
  • number: Solo acepta valores numéricos, con la opción de establecer rangos.
  • email: Campo diseñado para ingresar direcciones de correo, con validación automática.
  • date: Muestra un selector de calendario para elegir una fecha.
  • checkbox: Permite seleccionar o deseleccionar una opción.
  • radio: Similar al checkbox, pero solo permite seleccionar una opción dentro de un grupo.
  • file: Permite subir archivos desde el dispositivo del usuario.
  • color: Muestra un selector de colores para elegir un tono específico.
  • range: Permite seleccionar valores dentro de un rango definido, útil para opciones como volumen o brillo.

Atributos booleanos en Input

  1. checked : Marca un checkbox o radio button por defecto.
  2. disabled : Desactiva el campo de entrada, impidiendo la interacción.
  3. readonly : Hace que el campo sea de solo lectura.
  4. required : Indica que el campo es obligatorio en un formulario.
  5. autofocus : Hace que el campo reciba el foco automáticamente al cargar la página.
  6. multiple : Permite seleccionar múltiples archivos en un ``.
  7. novalidate : Indica que el formulario no debe validarse al enviarse.
